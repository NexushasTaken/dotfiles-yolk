# Dotfiles

## Clone and install

```bash
mkdir -p ~/.config/
git clone https://github.com/NexushasTaken/dotfiles-yolk.git ~/.config/yolk/
cd ~/.config/yolk/
# TODO: Not tested yet on my PC.

```
